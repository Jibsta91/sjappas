== Description ==

Best Listing - Smart and Powerful Directory & Listing WordPress Theme. If you are trying to create a directory website but are worried to give it the look the way you are portraying in your mind, Best Listing is the go-to theme for you. No matter which type of directory website you are going to build, Best Listing is designed in a way that can align with any design you can fantasize about. With this theme, you can easily create any kind of directory website for Restaurant, Automobile, Real Estate, Classified ads listings, Hotel and so many more.

Best Listing is developed in such a way so that people can scale up their lawyer directory website according to their needs. Literally, you can scale up to millions of directory listings using this awesome WordPress directory theme.

One of the most feature-rich components of Best Listing theme is its users friendliness. You can design and craft your each listing card as your wish. There are options to make individual listing card more personalized.

Using Best Listing theme, as an admin, you can customize, manage & configure the whole theme effortlessly. You can customize a single listing page, enable/disable sections based on your need, and configure the listing slider.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
